# Jumpin' Climbin'

Unityの学習及びインターンの課題用に作成したゲームです。
空中を自由に移動して、ひたすら上を目指しましょう。

##目的
ジャンプ台を利用してひたすら上に登っていくゲームです。

##操作方法
矢印キー：移動
スペースキー：ゲームの開始
Qキー：画面のキャプチャ（Imagesフォルダ以下に保存されます）

##ジャンプ台
####ノーマル
最も普遍的なジャンプ台。特徴は特に無し。

####赤
一番最初に登場する初心者用のジャンプ台。ノーマルよりも少し大きくて、ジャンプ力が高い。

####青
50mを過ぎると稀に登場。サイズは小さいが圧倒的なジャンプ力を誇る。
